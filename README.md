# Go Beyond Plus
Python Flask app for a side project in Unionbank Pecha Kucha night

### Steps to run
- Make sure you already have `virtualenv` installed
- run `pip install -r requirements.txt`
- run `bower install`
- `export FLASK_APP=app.py`
- `flask run`
